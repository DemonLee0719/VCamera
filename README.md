# VCamera
**This is an open source for android virtual camera.**  
*Just for protect your camera privacy.*


# Key Features
**1.Set Picture as Camera Video.**  
**2.Set Local Video as Camera Video.**  
**3.Set Network Stream Video as Camera Video.**
